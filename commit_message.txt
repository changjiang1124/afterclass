feat(speak_practice): Implement async topic loading and UI states

Refactors the topic selection page in the speak_practice app to load AI-generated topics asynchronously. This improves the initial page load time and user experience.

Key changes:
- Introduces an API endpoint (`/api/topics/`) to fetch topics dynamically.
- Implements a JavaScript-based `TopicLoader` on the frontend to handle the async request.
- Adds comprehensive UI states to the template:
  - Skeleton loading screen for initial view.
  - Animated loading indicator during the API call.
  - Detailed error state with retry and fallback options.
- Enhances the CSS to support the new UI states, including animations and dark mode compatibility.
- Moves the topic generation logic to the new API view, with robust error handling and fallback to static topics.
- Ignores temporary test files in `.gitignore`.
