# Product Overview

**Learn Chinese Perth** is an educational platform for a Chinese language school in Perth, Australia. The system provides comprehensive learning tools for Chinese language students including:

## Core Features
- **Typing Practice** (`typingchinese`) - Chinese character input training
- **Reading Comprehension** (`stories`) - Interactive story reading with TTS
- **Listening Practice** - Audio-based learning exercises  
- **Speaking Practice** (`speak_practice`) - Conversational AI scenarios
- **Assignments** - Structured learning tasks and quizzes
- **Name Generation** (`namegen`) - Chinese name generator tool
- **Pinyin Tools** (`pinyinit`) - Pinyin conversion and practice
- **Chatbots** - AI-powered conversational learning

## Target Users
- Chinese language students of all levels (beginner, intermediate, advanced)
- Students at Learn Chinese Perth school
- Self-directed learners seeking structured Chinese language practice

## Design Principles
- Professional, clean, and easy-to-use interface
- Accessibility-focused design
- Mobile-responsive layout
- Personalized learning experiences based on student profiles
