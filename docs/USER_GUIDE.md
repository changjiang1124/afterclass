## 功能概述 (Feature Overview)

增强聊天交互功能为中文学习者提供了更自然、更有效的口语练习体验，主要包括：

### 核心功能 (Core Features)
- **语音输入** (Voice Input): 通过麦克风录制中文语音，自动转换为文字
- **英文翻译辅助** (English Translation Assistance): 输入英文自动翻译为中文，帮助初学者
- **自动语音播放** (Automatic Voice Playback): AI回复自动朗读，学习正确发音
- **确认界面** (Confirmation Interface): 发送前确认和编辑识别结果
- **实时反馈** (Real-time Feedback): 处理状态和错误提示

### 支持的场景 (Supported Scenarios)
- 咖啡厅点餐 (Café Ordering)
- 自我介绍 (Self Introduction)
- 购物对话 (Shopping Conversations)
- 问路指引 (Asking for Directions)
- 日常交流 (Daily Conversations)
- 商务会话 (Business Conversations)

## 开始使用 (Getting Started)

### 1. 访问聊天功能 (Accessing Chat Feature)

1. 登录Learn Chinese Perth平台
2. 在仪表板中点击"Speaking Practice"
3. 选择您想要练习的场景
4. 点击"开始对话"进入聊天界面

### 2. 界面介绍 (Interface Introduction)

聊天界面包含以下主要元素：

- **对话区域** (Chat Area): 显示您和AI的对话历史
- **输入区域** (Input Area): 包含语音录制按钮和文本输入框
- **模式切换** (Mode Switch): 在语音输入和英文输入之间切换
- **导航按钮** (Navigation Buttons): 返回场景选择或重新开始对话
- **音频控制** (Audio Controls): 播放、暂停、重播AI回复

### 3. 权限设置 (Permission Setup)

首次使用语音功能时，浏览器会请求麦克风权限：

1. 点击浏览器地址栏中的麦克风图标
2. 选择"允许"或"Allow"
3. 如果意外拒绝，可在浏览器设置中重新启用

## 语音输入功能 (Voice Input Feature)

### 1. 录制语音 (Recording Voice)

#### 开始录制 (Start Recording)
1. 点击麦克风按钮 🎤
2. 按钮变为红色，显示"录音中..."
3. 清晰地说出您的中文内容
4. 录制时长建议控制在30秒以内

#### 停止录制 (Stop Recording)
1. 再次点击麦克风按钮
2. 系统开始处理您的语音
3. 显示"正在识别语音..."状态

### 2. 语音识别结果 (Speech Recognition Results)

录制完成后，系统会显示：
- **中文转录文本** (Chinese Transcription): 识别出的中文内容
- **英文翻译** (English Translation): 对应的英文翻译
- **确认选项** (Confirmation Options): 发送或重新录制

### 3. 确认和编辑 (Confirmation and Editing)

#### 确认界面功能 (Confirmation Interface Features)
- **编辑文本** (Edit Text): 点击中文文本可直接编辑
- **查看翻译** (View Translation): 了解内容的英文含义
- **重新录制** (Re-record): 如果不满意可重新录制
- **发送消息** (Send Message): 确认无误后发送

#### 编辑技巧 (Editing Tips)
- 可以修改识别错误的字词
- 添加标点符号提高表达准确性
- 编辑后会自动更新英文翻译

### 4. 语音输入最佳实践 (Voice Input Best Practices)

#### 录制环境 (Recording Environment)
- 选择安静的环境，减少背景噪音
- 确保麦克风工作正常
- 与麦克风保持适当距离（15-30厘米）

#### 说话技巧 (Speaking Tips)
- 语速适中，发音清晰
- 避免过快或过慢的语速
- 使用标准普通话发音
- 句子不要过长，建议分段表达

#### 常见问题处理 (Common Issue Handling)
- 如果识别不准确，尝试重新录制
- 方言或口音较重时，可能需要多次尝试
- 网络不稳定时可能影响识别速度

## 英文翻译辅助 (English Translation Assistance)

### 1. 切换到英文输入模式 (Switch to English Input Mode)

1. 点击输入区域的"English"按钮
2. 界面切换为英文输入模式
3. 文本输入框显示英文提示

### 2. 输入英文内容 (Input English Content)

#### 文本输入 (Text Input)
1. 在输入框中输入英文内容
2. 系统自动检测并准备翻译
3. 点击"翻译"按钮或按Enter键

#### 翻译处理 (Translation Processing)
- 显示"正在翻译..."状态
- 系统使用AI进行英中翻译
- 生成中文翻译和拼音标注

### 3. 翻译结果确认 (Translation Result Confirmation)

翻译完成后显示：
- **中文翻译** (Chinese Translation): 翻译后的中文内容
- **拼音标注** (Pinyin Annotation): 帮助发音的拼音
- **语音播放** (Voice Playback): 自动播放中文发音
- **确认选项** (Confirmation Options): 发送或重新翻译

### 4. 英文输入技巧 (English Input Tips)

#### 输入建议 (Input Suggestions)
- 使用简单、清晰的英文表达
- 避免过于复杂的语法结构
- 一次输入一个完整的想法或句子

#### 翻译优化 (Translation Optimization)
- 如果翻译不准确，尝试换个表达方式
- 可以在确认界面编辑中文翻译
- 利用拼音标注学习正确发音

## 自动语音播放 (Automatic Voice Playback)

### 1. AI回复语音播放 (AI Response Voice Playback)

#### 自动播放功能 (Automatic Playback Feature)
- AI回复后自动开始语音播放
- 使用Google TTS技术，发音标准
- 播放时显示音频控制按钮

#### 播放控制 (Playback Controls)
- **暂停** (Pause): 暂停当前播放
- **重播** (Replay): 重新播放当前内容
- **停止** (Stop): 停止播放并重置

### 2. 音频控制功能 (Audio Control Features)

#### 播放状态指示 (Playback Status Indicator)
- 播放中：显示波形动画
- 暂停：显示暂停图标
- 完成：显示重播按钮

#### 音频设置 (Audio Settings)
- 音量控制：使用系统音量设置
- 播放速度：标准语速，适合学习
- 音质：高质量MP3格式

### 3. 语音学习建议 (Voice Learning Suggestions)

#### 跟读练习 (Shadowing Practice)
1. 听完AI回复的语音
2. 暂停并尝试重复
3. 对比自己的发音
4. 多次练习直到熟练

#### 发音改进 (Pronunciation Improvement)
- 注意声调变化
- 学习正确的语音语调
- 重点练习难发音的词汇

## 故障排除 (Troubleshooting)

### 1. 语音功能问题 (Voice Feature Issues)

#### 麦克风无法使用 (Microphone Not Working)
**症状**: 点击录音按钮无反应或提示权限错误

**解决方案**:
1. 检查浏览器麦克风权限设置
2. 确认麦克风硬件工作正常
3. 尝试刷新页面重新授权
4. 检查系统音频设备设置

#### 语音识别不准确 (Inaccurate Speech Recognition)
**症状**: 识别出的文字与说话内容不符

**解决方案**:
1. 改善录音环境，减少噪音
2. 说话更清晰，语速适中
3. 使用标准普通话发音
4. 尝试分段录制较长内容

#### 语音识别超时 (Speech Recognition Timeout)
**症状**: 显示"语音识别超时"错误

**解决方案**:
1. 检查网络连接稳定性
2. 尝试录制较短的内容
3. 刷新页面重新尝试
4. 联系技术支持

### 2. 翻译功能问题 (Translation Feature Issues)

#### 翻译结果不准确 (Inaccurate Translation)
**症状**: 中文翻译与英文原意不符

**解决方案**:
1. 尝试使用更简单的英文表达
2. 在确认界面手动编辑中文内容
3. 分解复杂句子为简单句子
4. 使用常用词汇和表达

#### 翻译服务不可用 (Translation Service Unavailable)
**症状**: 显示"翻译服务暂时不可用"

**解决方案**:
1. 检查网络连接
2. 稍后重试
3. 切换到语音输入模式
4. 联系技术支持

### 3. 音频播放问题 (Audio Playback Issues)

#### 无法播放语音 (Cannot Play Audio)
**症状**: AI回复后没有语音播放

**解决方案**:
1. 检查系统音量设置
2. 确认浏览器音频权限
3. 尝试手动点击播放按钮
4. 刷新页面重新尝试

#### 音频播放卡顿 (Audio Playback Stuttering)
**症状**: 语音播放断断续续

**解决方案**:
1. 检查网络连接速度
2. 关闭其他占用带宽的应用
3. 等待音频完全加载后播放
4. 尝试重新生成语音

### 4. 一般性问题 (General Issues)

#### 页面加载缓慢 (Slow Page Loading)
**解决方案**:
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器
4. 联系技术支持

#### 功能按钮无响应 (Unresponsive Function Buttons)
**解决方案**:
1. 刷新页面
2. 检查JavaScript是否启用
3. 尝试使用其他浏览器
4. 清除浏览器缓存和Cookie

## 常见问题 (Frequently Asked Questions)

### Q1: 支持哪些浏览器？ (Which browsers are supported?)
**A**: 推荐使用Chrome、Firefox、Safari或Edge的最新版本。确保浏览器支持Web Audio API和麦克风访问。

### Q2: 语音识别支持方言吗？ (Does speech recognition support dialects?)
**A**: 系统主要针对标准普通话进行优化。方言可能影响识别准确性，建议尽量使用标准发音。

### Q3: 可以离线使用吗？ (Can it be used offline?)
**A**: 不可以。语音识别、翻译和TTS功能都需要网络连接才能正常工作。

### Q4: 录音内容会被保存吗？ (Are recordings saved?)
**A**: 录音文件仅用于实时转录，不会在服务器上永久保存。转录完成后音频数据会被删除。

### Q5: 如何提高语音识别准确性？ (How to improve speech recognition accuracy?)
**A**: 
- 使用清晰的标准普通话
- 在安静环境中录制
- 语速适中，不要过快或过慢
- 确保麦克风质量良好

### Q6: 翻译功能支持哪些语言？ (Which languages does translation support?)
**A**: 目前支持英文到中文的翻译。未来可能会添加更多语言支持。

### Q7: 遇到技术问题如何获得帮助？ (How to get help with technical issues?)
**A**: 
1. 首先查看本用户指南的故障排除部分
2. 尝试刷新页面或重启浏览器
3. 联系Learn Chinese Perth技术支持
4. 发送邮件至support@learnchineseperth.com.au

### Q8: 系统对话有长度限制吗？ (Is there a length limit for conversations?)
**A**: 为了保持最佳性能，每个对话会话有令牌限制。接近限制时系统会提示，建议开始新的对话。

### Q9: 可以保存对话记录吗？ (Can conversation records be saved?)
**A**: 对话记录会在会话期间保存，但不会永久存储。建议重要内容及时记录。

### Q10: 如何切换不同的练习场景？ (How to switch between different practice scenarios?)
**A**: 点击"返回"按钮回到场景选择页面，然后选择新的练习场景。
